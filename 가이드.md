## GIt 사용
1) GitHub 저장소 페이지 우측 상단 “Code” → “HTTPS” 복사 버튼 클릭(https://github.com/yeojeong735/CafeShareService.git)
2) PC에 터미널(CMD, Git Bash, 또는 Android Studio 내 내장 터미널)을 실행  
3) cd C:\CookAndroid\Project\CafeShareService  ->  원하는 폴더로 이동
4) git clone https://github.com/yeojeong735/CafeShareService.git  ->  저장소 클론


## Github에 변경 내용 저장 및 공유
1) 안드로이드 스튜디오 터미널 들어가기(view --> tool windows --> terminer)
2) git pull origin main  ->  내용 최신화
3) git add .  ->  작업 후 변경된 파일을 스테이지에 추가
4) git commit -m "수정한 작업 설명"  ->  변경 내용 커밋
5) git push origin main  ->  공동 저장소에 업로드


## 이미지
